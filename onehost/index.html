Lesson 1

HTML definition

Hypertext (hyper links) markup language (makes the elements)
HTML is the document containing all the information for the website.
Provides the structure; plain text.
The head is the thinking behind the website, the setup information, telling the browser information about the website without displaying it.
Also contains the meta tags that contains the information read by the browser. Followed by the link tag, ie. to the stylesheet, and the title tag being the name of the website.
The body is the clothes; everything that is to be displayed on the website.

CSS Definition

CSS makes the website look pretty. Provides visual enhancements to the plain text.
There are 3 ways to implement CSS styling:
INLINE STYLING: Adding the style in the individual line of code. Best used for email (HTML) in one file. Something smaller and singular.
INTERNAL STYLE TAGS: Found in the head. Increase the information talking to the browser. Not for larger scale projects.
EXTERNAL STYLESHEETS: Separate file for styling in the same folder. Stylesheets are added in the head link tag. These are used most commonly, and are easier to handly with a multi file project.

Style tags are made up of a selctor, property and value.
The selector selects which element a rule applies to; the property defines which aspect of an element is being controlled; the value defines the value a particular property will take according to the rule.
